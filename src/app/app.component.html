Angular i18n localization
Localizable content

With i18n you have a wide range of options to localize your application.

  • Texts
  • Dates
  • Numbers
  • Percentages
  • Currencies

You are also able to handle e.g. plurals or genders in the ICU format.

Hi, {{userName}}

Since your last registration on {{now | date:'shortDate'}}, {{minutes}} new articles have been published.

Dates

{{ now | date:'fullDate' }}

{{ now | date:'longDate' }}

{{ now | date:'short' }}

Numbers

{{ 100000000 | number }}

{{ 100000.23 | number }}

{{ 51.043534423 | number }}

Currencies

{{ 1000000 | currency }}

{{ 100.15 | currency }}

Percentages

{{ 1.2 | percent }}

{{ 0.034532 | percent:'0.2-2' }}

{{ 0.00011255 | percent:'0.3-5' }}

Plurals

Updated { minutes, plural, =0 {just now} =1 {one minute ago} other { {{minutes}} minutes ago } }

minutes: {{ minutes }}
Alternative texts

The author is {gender, select, m {male} f {female} o {other}}
